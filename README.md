# nonebot_plugin_sky

nonebot2的光遇每日任务及活动查询插件

</div>

# 开始使用

克隆本项目到本地，在bot.py中导入插件：

`nonebot.load_plugin(r"包路径\nonebot_plugin_sky")`


# 使用命令

在有bot的群内发送`/光遇每日攻略`来获取今日的攻略
数据来自微博@今天游离翻车了吗
